Delivery Order Report

Delivery Order Report based on Filter Criteria

This module add the Delivery Order Report. You can view the delivery order report by filtering based on Scheduled Date Range and Company

Usage

  1. Select Warehouse
  2. Select Delivery Order Report(Warehouse>Traceability)
  3. Select Scheduled From and To Date Range(It is mandatory)
  4. Select Company (It is optional)
  5. Click Print
  6. System will generate PDF Delivery Order Report
